## 🌐 Déploiement

### GitHub Pages (recommandé)
```bash
git init
git add .
git commit -m "Initial commit"
git push origin main
```
Activez Pages dans Settings → Pages → Deploy from main

URL : `https://votre-username.github.io/portfolio`

### Vercel
1. Connectez votre GitHub sur [vercel.com](https://vercel.com)
2. Importez le repo
3. Deploy

### Netlify
Glissez-déposez le dossier sur [netlify.com](https://netlify.com)
